## 📌 What Is CyberArsenal?

**CyberArsenal** is a curated personal toolkit of custom scripts, exploits, automation tools, and recon utilities developed during hands-on penetration testing, CTF challenges, and security research.

This folder serves as a living collection — growing as new tools are built and refined across various security disciplines including **network security, web exploitation, OSINT, post-exploitation, and defensive tooling**.

---

## 🧰 Categories

| Category | Description |
|---|---|
| 🔍 **Recon & OSINT** | Subdomain enumeration, IP intelligence, port scanners |
| 💉 **Web Exploitation** | Custom payload generators, fuzzer utilities |
| 🌐 **Network Tools** | Packet crafting, ARP spoof helpers, network scanners |
| 🛡️ **Defensive Scripts** | Log analyzers, anomaly detectors, alert automation |
| 🏴 **CTF Utilities** | Challenge-specific solvers and decode/encode helpers |
| ⚙️ **Automation** | Recon pipelines, batch scan wrappers |
