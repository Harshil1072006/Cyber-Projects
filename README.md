## 📌 Overview

Welcome to my cybersecurity portfolio repository! This workspace is dedicated to housing a diverse range of security tools, automated pipelines, offensive security scripts, and AI-powered defense mechanisms that I have developed. 

From full-scale Vulnerability Assessment platforms to custom browser extensions and Machine Learning security engines, this repository showcases hands-on experience in **DevSecOps, Penetration Testing, Threat Intelligence, and Security Automation**.

## 🗂️ Project Directory

Here is a summary of all the projects included in this repository:

### 🔍 Vulnerability Assessment & Pentesting
- **[`vapt_v1/`](./vapt_v1)**: An **AI-Powered VAPT Platform**. A full-stack solution combining industry-standard tools (Semgrep, ZAP, Nuclei, Trivy) with AI (LLaMA 3/Groq) for automated vulnerability scanning and professional report generation.
- **[`BurpKit/`](./BurpKit)**: A custom SQL Injection Testing Suite and web crawler for advanced web application security testing.
- **[`Web Application Vulnerability Scanner/`](./Web%20Application%20Vulnerability%20Scanner)**: Tools and frameworks dedicated to identifying and mitigating web-based vulnerabilities.

### 🛡️ Threat Detection & Intelligence
- **[`ThreatPulse/`](./ThreatPulse)**: A threat intelligence aggregation and processing platform designed to collect, enrich, and analyze indicators of compromise (IoCs).
- **[`Network Intrusion Detection System (IDS)/`](./Network%20Intrusion%20Detection%20System%20%28IDS%29)**: Core configurations and scripts for monitoring network traffic and identifying malicious activity in real-time.

### 🧠 Machine Learning in Security
- **[`VAPT_ML_Engine/`](./VAPT_ML_Engine)**: An ML-Based Vulnerability Predictor using trained models (like Random Forest) to detect insecure patterns and code-level vulnerabilities directly from source code snippets.

### ⚙️ DevSecOps & Pipelines
- **[`DevSecOps CI_CD Security Pipeline/`](./DevSecOps%20CI_CD%20Security%20Pipeline)**: Automated security checks, SAST, and DAST integrations designed to be seamlessly embedded into modern CI/CD workflows.

### 🛠️ Custom Utilities & Arsenal
- **[`CyberArsenal/`](./CyberArsenal)**: A personal collection of custom scripts, exploits, and utility tools built for specialized penetration testing tasks.
- **[`Odysseus/`](./Odysseus)**: Advanced security tooling and research.
- **[`PurePlay/`](./PurePlay)**: A browser extension utilizing content moderation techniques for safe browsing.
- **[`AutoFillPro/`](./AutoFillPro)**: A productivity browser extension demonstrating secure manifest V3 practices and local data handling.

3. Follow the installation and usage instructions provided.
