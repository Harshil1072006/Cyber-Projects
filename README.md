## 📌 Overview

Welcome to my cybersecurity portfolio repository! This workspace is dedicated to housing a diverse and advanced range of security tools, automated pipelines, offensive security scripts, and AI-powered defense mechanisms. 

Whether you are a newcomer looking to learn about DevSecOps, or a seasoned professional evaluating enterprise security solutions, you will find heavily documented, production-grade projects here.

### 🎯 Core Focus Areas:
- **Vulnerability Assessment (SAST/DAST/SCA)**

- **Machine Learning Applications in Cyber Security**
- **DevSecOps & CI/CD Security Automation**
- **Cloud Infrastructure (AWS, Terraform, Docker, Kubernetes)**

## 🚀 Pre-requisites & Getting Started

To get the most out of this repository, it is recommended to have the following set up on your local machine:

1. **Git** - To clone and navigate the repository.
2. **Python 3.10+** - Required for the majority of the ML and backend security engines.
3. **Docker & Docker Compose** - Used for containerized microservices and ELK stack deployments.
4. **Java 17+ & Maven** - Required for running and testing the Java/Spring Boot enterprise applications.
5. **Terraform & AWS CLI** - Needed for the Cloud Infrastructure automation projects.

**How to navigate:**
1. Pick a category from the directory below.
2. Navigate into the specific project folder.
3. Every project has its own `README.md` with specific installation steps, architecture diagrams, and usage examples.

## 🗂️ Project Directory

### 🛡️ Vulnerability Assessment & Application Security
Tools and frameworks dedicated to identifying, exploiting, and mitigating application vulnerabilities.

- **[`Java Enterprise App Security Assessment/`](./Java%20Enterprise%20App%20Security%20Assessment)**: A production-grade security testing tool for Spring Boot REST APIs featuring custom Semgrep SAST rules, OSV SCA, and targeted DAST payloads (SQLi, XXE, Deserialization).
- **[`Web Application Vulnerability Scanner/`](./Web%20Application%20Vulnerability%20Scanner)**: An advanced, automated vulnerability scanner tailored for web applications with robust payload injection engines and rich HTML reporting.
- **[`vapt_v1/`](./vapt_v1)**: An **AI-Powered VAPT Platform**. Combines industry-standard tools (Semgrep, ZAP, Nuclei, Trivy) with AI (LLaMA 3/Groq) for automated scanning and executive report generation.
- **[`React_Node.js Application Security Audit Toolkit/`](./React_Node.js%20Application%20Security%20Audit%20Toolkit)**: Specialized toolkit for auditing the security posture of MERN/PERN stack applications.
- **[`BurpKit/`](./BurpKit)**: A custom SQL Injection Testing Suite and web crawler built for advanced penetration testing.

### 🤖 Machine Learning in Security
Applying data science and AI models to predict and identify security threats.

- **[`VAPT_ML_Engine/`](./VAPT_ML_Engine)**: An ML-Based Vulnerability Predictor that uses trained models (e.g., Random Forest) to detect insecure patterns and code-level vulnerabilities directly from source code snippets.

### 🕵️ Threat Detection & Network Security
Monitoring, tracking, and halting malicious activity across networks.

- **[`ThreatPulse/`](./ThreatPulse)**: A threat intelligence aggregation platform designed to collect, enrich, and analyze Indicators of Compromise (IoCs).
- **[`Network Intrusion Detection System (IDS)/`](./Network%20Intrusion%20Detection%20System%20%28IDS%29)**: Core configurations and Python/Bash scripts for monitoring network traffic and identifying malicious packets in real-time.

### ⚙️ DevSecOps, Cloud & Infrastructure Automation
Pipelines and Infrastructure as Code (IaC) to securely deploy and manage environments.

- **[`DevSecOps CI_CD Security Pipeline/`](./DevSecOps%20CI_CD%20Security%20Pipeline)**: Automated security checks, SAST, and DAST integrations seamlessly embedded into modern GitHub/Jenkins CI/CD workflows.
- **[`Cloud Infrastructure Automation — AWS + Terraform/`](./Cloud%20Infrastructure%20Automation%20%E2%80%94%20AWS%20%2B%20Terraform)**: Scalable and secure cloud architecture deployment scripts using Terraform.
- **[`Containerised Microservices Deployment — Docker & Kubernetes/`](./Containerised%20Microservices%20Deployment%20%E2%80%94%20Docker%20%26%20Kubernetes)**: Enterprise-grade deployment templates for orchestrating secure microservices.
- **[`Cloud VM Provisioning & Basic IaC Setup/`](./Cloud%20VM%20Provisioning%20%26%20Basic%20IaC%20Setup)**: Baseline scripts for spinning up secure, hardened virtual machines.

### 📊 Log Analysis & Monitoring
Ensuring visibility and alerting for enterprise systems.

- **[`Centralized Log Analysis Pipeline — ELK Stack/`](./Centralized%20Log%20Analysis%20Pipeline%20%E2%80%94%20ELK%20Stack)**: A full Elasticsearch, Logstash, and Kibana setup for aggregating security logs.
- **[`Automated Log Aggregation & Incident Alerting Pipeline/`](./Automated%20Log%20Aggregation%20%26%20Incident%20Alerting%20Pipeline)**: Scripts designed to parse server logs and trigger automated alerts on suspicious activity.
- **[`Linux Server Monitoring & Alerting Dashboard/`](./Linux%20Server%20Monitoring%20%26%20Alerting%20Dashboard)**: Visual dashboard integrations for tracking system health and unauthorized access attempts.
- **[`Linux Server Health Monitoring System/`](./Linux%20Server%20Health%20Monitoring%20System)**: Lightweight Bash and Python agents for host-based metrics collection.

### 🛠️ Custom Utilities & Arsenal
Personal scripts, extensions, and DevOps productivity tools.

- **[`CyberArsenal/`](./CyberArsenal)**: A collection of custom scripts, payloads, and utility tools built for specialized penetration testing tasks.
- **[`Infrastructure Automation with Bash, Python & CI/`](./Infrastructure%20Automation%20with%20Bash%2C%20Python%20%26%20CI)**: Everyday automation scripts for maintaining environments.
- **[`DevOps Things/`](./DevOps%20Things)**: Miscellaneous DevOps utilities and configuration snippets.
- **[`PurePlay/`](./PurePlay)**: A browser extension utilizing content moderation techniques for safe browsing.
- **[`AutoFillPro/`](./AutoFillPro)**: A productivity browser extension demonstrating secure manifest V3 practices and local data handling.
